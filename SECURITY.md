# Security Policy

## Prototype scope

This public demonstration is intended for synthetic or approved non-confidential data only. Do not upload proprietary vehicle data, personal information, credentials, export-controlled information, or confidential engineering records.

## Reporting a vulnerability

Please report security issues privately through the repository owner's GitHub profile rather than opening a public issue with exploit details.

## Known security gaps

The prototype does not provide authentication, RBAC, tenant isolation, malware scanning, enterprise audit logging, or formal data-retention controls. See `LIMITATIONS.md` before considering any internal pilot.
